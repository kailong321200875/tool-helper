tool-helper

介绍

收集整理常用的工具函数，方便统一维护管理。同时也可以当做一个学习 rollup + typescript 的途径。

欢迎大家贡献代码~

安装

npm

 
npm i tool-helper -S

pnpm

 
pnpm i tool-helper -S

yarn

 
yarn add tool-helper

使用

ESM

 
import { isNumber } from 'tool-helper'

isNumber('test')

CJS

 
const tools = require('tool-helper')

tools.isNumber('test')

浏览器支持

支持现代浏览器及IE11，11以下的未测试。
